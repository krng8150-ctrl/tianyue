📘 新手完整教學

新手起手流程

1. 進入遊戲後，先找傳送師傳送到 30等練功區
2. 開啟內掛，先穩定把角色練到 50等左右
3. 50等後前往 傲慢之塔 收集 傲慢幣
4. 存夠傲慢幣後，可向 NPC 購買基礎裝備，或進行武器製作
5. 中期開始補強：耳環 / 戒指 / 項鍊 / 腰帶
6. 後期發展方向：新傲慢幣 → 高階防具 → 紋樣系統 → 轉生系統 → 龍珠 / 神龍武器

新手避坑重點

✔ 不要一進服就想衝最終裝，先把等級、掛機穩定度、傲慢幣來源建立起來
✔ 本服掛機掉寶多數為直接吸入身上，不一定會看到撿取動作
✔ 搜怪範圍不要設太小，太小容易原地亂跑
✔ 先有基礎裝，再去拚高階防具與後期系統，成長會比較穩

中後期成長方向

✔ 裝備升階：把基礎裝備慢慢強化到高階
✔ 傲慢幣 / 新傲慢幣：本服重要成長核心
✔ 高階防具：影響後期生存、續戰、PVP能力
✔ 紋樣系統：給角色額外數值與屬性成長
✔ 龍珠系統：後期取得神龍武器的重要流程
✔ 轉生系統：進一步拉開角色強度差距
